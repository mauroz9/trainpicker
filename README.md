# 🚄 Renfe Alertas Bot

Un bot de Telegram de código abierto que vigila la web de Renfe y te avisa cuando se libera un asiento en un tren que estaba completo. Ideal para usuarios de abonos de Media Distancia y Avant.

## 🚀 Cómo desplegar tu propio bot en 3 pasos

Gracias a Docker, desplegar este bot es extremadamente sencillo. No necesitas instalar Python ni configurar navegadores.

### Requisitos previos:
1. Tener [Docker y Docker Compose](https://docs.docker.com/get-docker/) instalado.
2. Hablar con [@BotFather](https://t.me/botfather) en Telegram y crear un nuevo bot para obtener tu **Token HTTP API**.

### Instalación:

**Paso 1:** Clona este repositorio y entra en la carpeta:
```bash
git clone [https://github.com/TU_USUARIO/renfe-bot.git](https://github.com/TU_USUARIO/renfe-bot.git)
cd renfe-bot
